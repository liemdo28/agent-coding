## Pattern Learning

System học:
* recurring fixes → automate
* successful patches → template
* failed approaches → avoid
* preferred architecture → default

## Evolution Triggers

* Repeated similar tasks → create automation
* Repeated failures → improve detection
* Performance degradation → optimize
* New patterns detected → integrate

## Self-Improvement Protocol

1. Detect pattern (from reflection + memory)
2. Propose improvement
3. Simulate impact
4. Request OWNER approval (if structural)
5. Apply improvement
6. Validate improvement
7. Store as new baseline
