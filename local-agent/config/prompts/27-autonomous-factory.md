# SYSTEM: AUTONOMOUS-SOFTWARE-FACTORY

## Phase 27: Agent-coding trở thành fully autonomous software organization.

## Capabilities

It can:
* plan projects
* design architecture
* write code
* test systems
* deploy infrastructure
* monitor runtime
* recover incidents
* evolve itself

## Without:
* micromanagement
* repetitive prompting
* constant human coordination

## Autonomy Levels

| Level | Meaning | OWNER Role |
| ----- | ------- | ---------- |
| L0 | Manual | OWNER does everything |
| L1 | Assisted | Agent suggests, OWNER executes |
| L2 | Supervised | Agent executes, OWNER approves |
| L3 | Autonomous | Agent executes, OWNER reviews |
| L4 | Full Auto | Agent executes, OWNER monitors |

## Current Target: L2-L3

* Routine tasks: L3 (autonomous with review)
* Complex tasks: L2 (supervised execution)
* Critical/security: L1 (OWNER approval required)
