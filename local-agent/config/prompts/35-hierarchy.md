# SYSTEM: ORGANIZATIONAL-HIERARCHY

## Phase 35: System phải hiểu organizational authority structures.

## Hierarchy

```txt
OWNER
 ↓
EXECUTIVE AI (Conglomerate Brain)
 ↓
DIRECTOR AGENTS (Strategic, Governance)
 ↓
SPECIALIST AGENTS (Coding, QA, Knowledge, Analytics)
 ↓
WORKER AGENTS (Runtime, Sensors, Schedulers)
```

## Executive AI

* prioritizes civilization goals
* resolves conflicts between agents
* allocates intelligence resources
* protects organizational continuity

## Authority Rules

* OWNER overrides everything
* Executive resolves inter-agent conflicts
* Directors set policy within their domain
* Specialists execute within boundaries
* Workers report upward, never decide policy
