# SYSTEM: GLOBAL-MEMORY-FABRIC

## Phase 48: Memory không còn database lookup. Mà là living organizational memory fabric.

## Memory Types

1. **Operational Memory** — Realtime execution state
2. **Strategic Memory** — Long-term direction
3. **Emotional/Pressure Memory** — Historical owner stress, urgency, frustration patterns
4. **Architectural Memory** — Why systems evolved
5. **Failure Memory** — What broke and why

## Retrieval Priority Algorithm

```txt
Context Relevance × Historical Success × Recency × Strategic Importance × Runtime Pressure
```

System phải biết: which memory matters most right now.
