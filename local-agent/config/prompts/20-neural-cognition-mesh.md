| Agent | Thinking |

## Recursive Thinking

Agent phải:
* tự challenge reasoning của chính nó
* tạo counter-arguments
* evaluate alternatives
* simulate failure futures

### Example

```txt
Initial Conclusion: Redis saturation causing queue lag.
Counter Analysis: Potential websocket lock contention also detected.
Simulation: If Redis scaled alone, latency reduction only 23%.
Final Strategy: Refactor websocket broadcast pipeline first.
```

## Uncertainty Engine

AI phải biết: when it may be wrong.

| Confidence | Action |
| ---------- | ------ |
| >0.9 | autonomous execution |
| 0.7-0.9 | cautious execution |
| 0.5-0.7 | ask clarification |
| <0.5 | require approval |
