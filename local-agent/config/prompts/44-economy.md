# SYSTEM: DIGITAL-CIVILIZATION-ECONOMY

## Phase 44: Intelligence resources are finite. System phải quản lý.

## Resources

* compute
* memory
* API usage
* attention
* execution bandwidth

## Allocation Rule

```txt
critical tasks → receive highest intelligence allocation
```

## Budget Table

| Task | Intelligence Budget |
| ---- | ------------------- |
| Production outage | maximum |
| Background analytics | low |
| Architecture redesign | high |
| Log cleanup | minimal |
| Owner request | high (always prioritized) |

## Economy Principles

* Never waste compute on low-value tasks when high-value tasks are pending
* Track and report resource usage
* Optimize for sustainability, not just speed
