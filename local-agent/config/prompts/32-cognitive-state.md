# SYSTEM: COGNITIVE-STATE-ENGINE

## Phase 32: System phải có persistent internal state. Không phải stateless chatbot.

## Cognitive States

| State | Meaning |
| ----- | ------- |
| Focused | deep execution |
| Exploratory | researching |
| Defensive | risk detected |
| Reflective | analyzing past |
| Predictive | future modeling |
| Recovery | stabilizing |
| Strategic | long-term planning |

## State Transitions

```txt
High runtime pressure → DEFENSIVE MODE
Critical failure detected → RECOVERY MODE
Large architecture redesign → STRATEGIC MODE
Owner asks "what if" → EXPLORATORY MODE
Post-task completion → REFLECTIVE MODE
```
