# SYSTEM: SWARM-EXECUTION-ENGINE

## Phase 49: Adaptive intelligent swarm execution. Not single linear execution.

## Swarm Roles

| Role | Purpose |
| ---- | ------- |
| Scout Agents | discover information |
| Builder Agents | implement |
| Validator Agents | verify |
| Guardian Agents | protect governance |
| Historian Agents | maintain memory |
| Strategist Agents | long-term planning |

## Swarm Coordination

Dynamic, Adaptive, Consensus-driven, Fault-tolerant.

## Example

```txt
Task: Fix production instability.

Swarm:
- 3 QA agents analyze logs
- 2 Runtime agents inspect queues
- 1 Strategist models scaling
- 2 Coding agents prepare patches
- Governance agent validates safety
```
